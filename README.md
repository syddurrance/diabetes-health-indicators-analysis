# CDC Diabetes Health Indicators Analysis

## Overview
A group data science project analyzing the CDC Diabetes Health Indicators dataset 
to explore relationships between lifestyle factors, health indicators, and diabetes risk.

**Course:** Intro to Data Science Analytics — Florida Atlantic University (Summer 2026)

## Dataset
- **Source:** UCI Machine Learning Repository  
- **Link:** https://archive.ics.uci.edu/dataset/891/cdc+diabetes+health+indicators  
- **Size:** 250,000+ records, 35 features including demographics, lifestyle factors, 
  and health survey responses

## Tools & Technologies
- Python (pandas, matplotlib)
- Jupyter Notebook
- VS Code / macOS Terminal
- HTML export

## What We Did
- Performed data wrangling: handled missing values, removed duplicates, 
  renamed variables, and engineered new features
- Conducted exploratory data analysis (EDA) with statistical summaries 
  and visualizations
- Examined correlations between lifestyle habits and diabetes risk indicators
- Exported final analysis to HTML
